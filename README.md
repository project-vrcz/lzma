# LZMA SDK

This repository contains a snapshot of [LZMA SDK v26.00](https://www.7-zip.org/sdk.html) and pre-complied binary.

## Pre-complied binary

Currently we build LZMACon (`lzma.exe` `lzma-sdk\CPP\7zip\Bundles\LzmaCon`) for those platform:

- Windows x64
- Linux x64

Pre-complied binary are built on [Github Action](https://github.com/project-vrcz/lzma/actions) and release at [Github Release](https://github.com/project-vrcz/lzma/releases).
